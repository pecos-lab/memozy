# Memozy QA 테스트 케이스

## 디렉토리 구조

```
qa_test_case/
├── README.md              # 이 파일 (가이드)
└── v1.0.0/                # 버전별 테스트 케이스
    ├── TC-001_메모_CRUD.md
    ├── TC-002_카테고리_필터.md
    ├── TC-003_설정_테마.md
    ├── TC-004_설정_언어.md
    ├── TC-005_설정_초기화.md
    ├── TC-006_네비게이션.md
    └── TC-007_UI_엣지케이스.md
```

## 테스트 케이스 작성 규칙

| 필드 | 설명 |
|------|------|
| TC-XXX | 테스트 케이스 ID |
| 우선순위 | P0(필수) / P1(높음) / P2(중간) / P3(낮음) |
| 상태 | ⬜ 미실행 / ✅ 통과 / ❌ 실패 / ⚠️ 조건부 통과 |

## 버전별 추가 방법

새 버전 출시 시 `v{버전명}/` 폴더를 새로 생성하고,
이전 버전 TC를 기반으로 변경사항 반영하여 업데이트할 것.
