| 001-006 | P1 | 저장 후 메모 탭으로 이동 | 설정 탭에 있음 | 설정 탭에서 + 버튼 → 메모 작성 → 저장 | 메모 탭(홈)으로 이동 | ⬜ |
| 001-007 | P1 | 저장 후 메모 탭으로 이동 | 메모 탭에 있음 | 메모 탭에서 + 버튼 → 메모 작성 → 저장 | 메모 탭(홈)으로 이동 | ⬜ |

| 001-009 | P2 | 내용 공백만 입력 | 메모 추가 화면 | 제목 입력, 내용에 스페이스만 입력 | 저장 버튼 비활성화 (공백은 빈 값 취급) | ⬜ |
| 001-010 | P2 | 메모 여러 개 연속 추가 | 앱 실행 | 메모 5개 연속 저장 | 모든 메모 목록에 정상 표시 | ⬜ |
