# Popup

사용자의 즉각적인 주의와 응답이 필요한 상황에서 사용하는 모달 요소. 현재 작업을 중단시키고 팝업 내용에 집중하도록 하며, 필수적인 결정이나 확인이 필요한 경우에 사용한다.

---

## Anatomy

1. **Navigation** — 타이틀/닫기 영역
2. **Contents area** — 콘텐츠 영역
3. **Scrim** — 딤 배경
4. **Action area** — 액션 버튼 영역

---

## Size

모든 사이즈는 **고정(Fixed)** / **변동(Hug)** 두 가지 타입 사용 가능.

| Size | Fixed | Hug (최대) | Padding |
|------|-------|-----------|---------|
| **Medium** | 400×480px | 400×760px | 20px |
| **Large** | 480×560px | 480×760px | 24px |
| **Xlarge** | 560×640px | 560×760px | 32px |

- Fixed: 콘텐츠가 영역 초과 시 스크롤 발생
- Hug: 콘텐츠가 최대 높이까지 늘어나다 스크롤 발생

## Contents 배치

| 배치 | 설명 | 사용 시점 |
|------|------|----------|
| **Center align** | 화이트 스페이스 극대화, 주목도 높음 | 중요 알림, 주요 상태, 짧은 텍스트 + 작은 모달 |
| **Left align** | 좌→우/상→하 가독성 좋음 | 비교적 긴 콘텐츠 |
| **With image** | Navigation의 Floating 타입 사용 | 이미지와 함께 사용, 컬러 값 유동 조정 가능 |

## 가이드라인

- **Do**: 일정 기간 동안 띄우는 모달일 경우 '일주일 동안 보지 않기' 등 해당 정보를 보지 않을 수 있는 권한 제공 권장
- **Don't**: 닫기 버튼과 [X] 버튼을 중복 배치하지 않음
