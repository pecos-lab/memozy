## 기본 원칙
- 기존 기능을 절대 제거하거나 덮어쓰지 말 것
- 요청한 부분만 최소한으로 수정할 것
- 수정 전 반드시 기존 코드 전체 구조 파악 후 작업할 것
- 확신이 없으면 수정하지 말고 먼저 질문할 것
- 작업 완료 후 기존 기능 목록과 유지 여부를 체크리스트로 보고할 것

## 코드 수정 규칙
- 파일 전체를 재작성하지 말 것
- 변경이 필요한 함수/컴포넌트만 수정할 것
- 기존 변수명, 함수명 유지할 것
- 기존 import 제거하지 말 것

## 작업 순서
1. 기존 코드 구조 파악
2. 변경 범위 최소화
3. 수정 후 기존 기능 유지 여부 확인
